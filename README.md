# TIL-record란?
블로그 운영을 위해 매일 실험한 내용, 회고, 코드 등을 정리한 레포입니다.

## 사용 목적
- 글 올리는 과정을 **공유**하고자 합니다.
- 글 하나당 2~6일씩 소요되는 블로그 대신 매일 작성하여 **글 작성 주기를 세분화**하기 위해서입니다.(1일 1커밋 목표)
- git, github 사용에 익숙해지기 위함입니다.

## 블로그 주소: [티스토리 블로그](https://hyo6490.tistory.com/)

### 블로그 카테고리:
1. **AI 기초**: CNN-based, transformer-based 모델의 개발시 문제사항을 정리(발전과정에 따라)하고, 이를 해결하기 위한 실험 진행.
2. **AI 의료**: 의료 데이터를 직접 AI에 적용한 실험. -> 업로드 예정
3. **AI 인프라**: AI를 서비스화 하기위해 필요한 서빙 기술(추론 최적화, 병렬처리 기법)에 대한 정리. -> 업로드 예정
